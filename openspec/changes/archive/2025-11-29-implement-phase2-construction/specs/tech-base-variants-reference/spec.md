## ADDED Requirements

### Requirement: Variant Documentation

Tech base variants SHALL be documented for reference.

#### Scenario: XL Engine variants

- **WHEN** referencing XL engine differences
- **THEN** IS XL uses 3 side torso slots
- **AND** Clan XL uses 2 side torso slots
- **AND** weight is identical

#### Scenario: Double heat sink variants

- **WHEN** referencing double heat sink differences
- **THEN** IS double uses 3 external slots
- **AND** Clan double uses 2 external slots

### Requirement: Slot Difference Pattern

The "3-2 Pattern" SHALL be documented for slot differences.

#### Scenario: Pattern application

- **WHEN** applying tech base slot differences
- **THEN** IS typically uses 3 slots for spread components
- **AND** Clan typically uses 2 slots due to miniaturization
