# Typography

MekStation uses a clear typographic hierarchy inspired by military HUD displays and the COMP/CON aesthetic.

## Font Stack

The application uses system fonts for optimal performance:

```css
font-family: Arial, Helvetica, sans-serif;
```

## Base Font Size

The base font size is configurable via user settings:

```css
--font-size-base: 16px; /* Default */
```

## Text Classes

### Display Text

Large, spaced typography for major headers:

<div className="bg-surface-deep p-6 rounded-lg space-y-4">
  <div className="text-display">COMPENDIUM</div>
  <div className="text-display-sm">MECHS</div>
</div>

```html
<div class="text-display">COMPENDIUM</div>
<div class="text-display-sm">MECHS</div>
```

### Section Headers

Bold uppercase for section divisions:

<div className="bg-surface-deep p-6 rounded-lg">
  <div className="text-section-header">Equipment Loadout</div>
</div>

```html
<div class="text-section-header">Equipment Loadout</div>
```

### Category Labels

Small uppercase for navigation and categorization:

<div className="bg-surface-deep p-6 rounded-lg">
  <div className="text-category-label text-amber-400">BattleMech</div>
</div>

```html
<div class="text-category-label text-amber-400">BattleMech</div>
```

### Subtitles

Secondary descriptive text:

<div className="bg-surface-deep p-6 rounded-lg">
  <div className="text-subtitle">Configure your mech's armor distribution</div>
</div>

```html
<div class="text-subtitle">Configure your mech's armor distribution</div>
```

## Best Practices

1. **Use semantic hierarchy** - H1 for page titles, H2 for sections, etc.
2. **Maintain contrast** - Use primary text for important content
3. **Be consistent** - Use the same patterns throughout the app
4. **Use monospace for data** - Technical stats and numerical values
5. **Uppercase sparingly** - Reserve for labels and emphasis
