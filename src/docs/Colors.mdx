# Color System

MekStation uses a semantic color system built on CSS custom properties with Tailwind CSS integration.
Colors are organized into categories based on their purpose.

Used for backgrounds and container elements. These create the layered depth of the dark theme.

## Usage Guidelines

### Layering
Use surface colors to create depth:
- `surface-deep` for the deepest background
- `surface-base` for cards and containers
- `surface-raised` for elevated elements (hover states, dropdowns)

### Text Hierarchy
- **Primary**: Main content, headings
- **Secondary**: Supporting text, labels
- **Muted**: Placeholders, disabled states

### Borders
- **Subtle**: Dividers, subtle separation
- **Default**: Standard borders
- **Strong**: Emphasis, focused elements
